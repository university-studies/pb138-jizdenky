Dependencies:
 * JDK 7/8
 * maven

Run web application
 * cd trunk
 * mvn install
 * cd jizdenky-web
 * mvn tomcat7:run
 -> localhost:8080/jizdenky-web


Sprava v docbook je umiestnena v adresari doc/
    pre vygenoravanie PDF pouzite `make`, musite mat nainstalovane pomocne
    nastroje (xsltproc a fop).
